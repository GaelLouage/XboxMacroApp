Getting Started:

    Running the Application and Connecting the Controller:
        Launch the application and connect your controller to a USB port.

Adding a Program:

    Supported File Types: Only .txt, .exe, and .png files are currently supported.

    To add a program, follow these steps:
        Click on the plus sign.
        Select your desired program from the list.
        After adding it, select the program and press the 'Key' button.
        Choose the corresponding Xbox button you want to map for running the selected program.

Deleting a Program:

    To delete a program:
        Select the program you want to remove and delete it.

Checking Controller Buttons:

    To check the controller buttons:
        Press the icon located at the top center of the application.
